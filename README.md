DATA PROFESSIONAL SURVEY DASHBOARD

PROJECT SUMMARY

* This project showcases an interactive Power BI dashboard built using a public Data Professional Survey dataset.
* The dashboard analyzes trends related to job roles, salaries, programming language preferences, career entry difficulty,
and satisfaction levels among data professionals.

* The project demonstrates practical skills in data analysis, visualization design, and insight communication using Power BI.

BUSINESS QUESTIONS ADDRESSED

* How salaries vary across different data roles

* Which programming languages are most commonly used

* Distribution of data professionals by country

* Difficulty level of entering the data field

* Satisfaction with salary and work-life balance

KEY INSIGHTS

* Data Scientist roles show the highest average salary among surveyed positions

* Python is the most preferred programming language across roles

* Many respondents report moderate to high difficulty entering the data industry

* Work-life balance satisfaction is higher than salary satisfaction overall

DATA SOURCE

* Public survey dataset sourced from the web

* Non-confidential data used strictly for learning and portfolio purposes

* Contains responses from data professionals across multiple countries and roles
